BaryPos : find the barycenter of multiples position on a globe

Find the point with the smallest sum of the distance to all the points placed. (the distances aresquared to avoid that when there's 2 point the "barycenter" can be everywhere between them) The method to find it is randomnized, so I need some time to compute the probabilites and fine tune it.

To find this point only in lands is under improvment, actually there's a method, but this method have low chances to find it if it's on a small island. (not with the updated actual one) This method also need imporvments in the file size (so the precision) of the map of lands in earth.

A third method is under development, searching barycenter by using distance matrices api (barycenter in this case will be the nearest destination to travel by car).
